# hospital-management # Medical & Hospital Control Panel 🏥

An interactive, frontend-based healthcare management platform crafted with HTML5, CSS3, and modern JavaScript. This system streamlines data management by handling patient information, secure portal entries for staff, and specialized interfaces, all backed by seamless browser storage.

## 🛠️ Key Capabilities

* **Role-Based Authentication:** Distinct and protected login portals for Administrators, Medical Staff, and Front Desk Executives.
* **Patient Records Hub:** Smooth interface to register, monitor, and update healthcare records instantly.
* **Persistent Session Data:** Uses web storage so no data is lost when reloading the application.
* **Tailored Working Views:** Unique layouts and operation centers for doctors and receptionists to organize tasks.
* **Fully Responsive Dynamic Pages:** A multi-page layout optimized to look great across devices.

---

## 📂 Architecture & File Overview

1. `login.html` – Portal entrance for existing portal members.
2. `signup.html` – Registration form to set up new system profiles.
3. `patients.html` – Central panel to filter and oversee the client list.
4. `receptionist-dashboard.html` – Operational center built for frontline desks.
5. `style.css` – Global stylesheet containing core layout properties and configurations.
6. `js/auth.js` – Logic engine responsible for session authorization and entry controls.
7. `js/storage.js` – Component handling secure data transactions with the browser's storage layer.

---

## 💻 Technical Stack

* **HTML5:** Semantic architecture for crisp rendering.
* **CSS3:** Adaptive layout frameworks and custom design styles.
* **JavaScript:** Dynamic scripting powered by ES6 features and LocalStorage API.
